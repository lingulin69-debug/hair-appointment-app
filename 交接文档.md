# 美髮院 H.SALON 開發進度

## 已完成組件

### ✅ Navbar（導覽列）
- 路徑：`src/components/UI/Navbar.tsx`
- 功能：
  - 左側：標題 + 手機版計算機
  - 中間：4 個視圖切換按鈕（預約日曆、顧客目錄、服務項目、營收休假）
  - 右側：電腦版計算機

## 項目結構

```
hair/
├── index.html              # 入口 HTML（含 Tailwind CDN + 自定義樣式）
├── src/
│   ├── main.tsx           # React 入口
│   ├── App.tsx            # 主應用（含全屏佈局 + Navbar）
│   └── components/
│       └── UI/
│           └── Navbar.tsx # 導覽列組件
├── package.json           # 依賴項已安裝
└── vite.config.ts         # Vite 配置
```

## 如何運行

```bash
cd hair
npm install    # 首次需安裝依賴
npm run dev    # 啟動開發伺服器
```

## 依賴項

- React 18.2.0
- lucide-react 1.7.0（圖標庫）
- Firebase 10.7.1
- Tailwind CSS（CDN）
